# Folder Map

Use this portable layout for a shared AI workspace.

## Top level

- `AGENTS.md` for shared rules
- `MEMORY.md` for stable decisions and project state
- `workstations/` for domain-specific folders
- `.agents/skills/` for shared skills
- `raw/` for original inputs
- `wiki/` for reusable knowledge
- `logs/` for dated execution records
- `Resources/` for reference files and templates

## Per-domain workspace

Each domain folder can contain:

- `CLAUDE.md` for local agent instructions
- `MEMORY.md` for domain memory
- `raw/` for domain-specific source material
- `wiki/` for domain knowledge pages
- `logs/` for dated work records
- `Resources/` for templates and references

## Classification rule

- raw = unprocessed or source material
- wiki = reusable knowledge and distilled pages
- MEMORY = stable decisions, preferences, project state
- logs = what happened on a given date
- Resources = templates, examples, checklists, source references
