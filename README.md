# Shared Vault Workflow

This repo is a portable, secret-safe starter pack for shared Obsidian-based AI workspaces.

## Start here

If you are a human:

1. Read [START_HERE.md](START_HERE.md)
2. Read [references/onboarding-zh.md](references/onboarding-zh.md)
3. If you plan to share this repo, read [references/redaction.md](references/redaction.md)

If you are an AI agent:

1. Read [AGENT_PROMPT_zh.md](AGENT_PROMPT_zh.md)
2. Follow [SKILL.md](SKILL.md) if your environment supports skills
3. Otherwise follow [references/onboarding-zh.md](references/onboarding-zh.md)

## What is inside

- `SKILL.md`: portable workflow for skills-aware agents
- `AGENT_PROMPT_zh.md`: direct bootstrap prompt for any agent
- `START_HERE.md`: short human-facing onboarding page
- `references/`: folder layout, maintenance rules, redaction, export checklist

## What this repo is for

Use these files to set up a transferable multi-device workspace with clear raw/wiki/MEMORY/log rules and secret-safe sharing.
