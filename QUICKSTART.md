# Quickstart

If you are human, read:

1. `START_HERE.md`
2. `references/onboarding-zh.md`
3. `references/redaction.md`
4. `references/logs-safety.md`
5. `references/resources-checklist.md`

If you are an AI agent, read:

1. `AGENT_PROMPT_zh.md`
2. `SKILL.md`
3. `references/folder-map.md`
4. `references/maintenance.md`

Then use these rules:

- `raw/` = original material
- `wiki/` = reusable knowledge
- `MEMORY.md` = durable decisions
- `logs/` = dated activity
- `Resources/` = templates and references

Before sharing the repo, run `scripts/check-secrets.sh`.
